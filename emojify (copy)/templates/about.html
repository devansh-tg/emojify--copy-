About This Project

üéØ Project Overview

This is an AI-powered real-time emotion detection system that uses Deep Learning to classify facial expressions into 7 different emotions: Angry, Disgust, Fear, Happy, Neutral, Sad, and Surprise.

üß† Technology Stack

  • Backend: Flask (Python Web Framework)
  • Deep Learning: TensorFlow / Keras
  • Computer Vision: OpenCV
  • Frontend: HTML5, CSS3, JavaScript
  • Model: Convolutional Neural Network (CNN)

üìä Model Architecture

  • Input: 48x48 grayscale images
  • Architecture: 3 Convolutional blocks with BatchNormalization
  • Parameters: ~2 million trainable parameters
  • Training: 100 epochs with early stopping
  • Platform: Kaggle GPU (T4 x2)
  • Dataset: 35,000+ facial expression images
  • Accuracy: 63.35% (Good for 7-class classification)

‚ú® Features

  • Real-time emotion detection from webcam
  • Live probability distribution for all emotions
  • Performance metrics (inference time, FPS)
  • Emotion history tracking
  • Responsive web interface
  • REST API for integration

üöÄ How It Works

  1. Camera captures real-time video feed
  2. Haar Cascade detects faces in each frame
  3. Face region is extracted and preprocessed (48x48 grayscale)
  4. CNN model predicts emotion probabilities
  5. Results are displayed with confidence scores
  6. System updates statistics and history

üìà Performance

  • Inference Speed: 20-30ms per prediction
  • Real-time FPS: 25-35 frames per second
  • Model Size: ~8 MB
  • Accuracy: 63.35% (7-class classification)
